How does javascript work?

JavaScript is an interpreted, single threaded and client-side scripting language. It can handle one task at a time or, in other words, a piece of code at a time. The web browser loads a web page of JS code then the codes are passed into a program called a compiler, which translates into bytecode that the machine understands and can execute the code. The DOM presents a live view of the web page to your JavaScript code.The JavaScript is executed in the order the code is written.

What does javascript event loop do?

JavaScript executes all operations on a single thread but it use many smart data structures. it behae multi-threading. rather they are built on top of the core JavaScript language in the browser. JavaScript has a runtime model based on an event loop, which is responsible for executing the code, collecting and processing events. The function stack is a function which keeps track of all other functions executed in run time.
